# Contributing

Thanks for considering a contribution.

## Goals

This repository stores reusable memory/rules and helper documentation for AI
tooling. Contributions that improve clarity, correctness, and maintainability
are welcome.

## How to Contribute

1. Fork the repository and create a topic branch.
2. Make focused changes with clear commit messages.
3. Open a pull request describing what changed and why.

## Preferred, Not Required

If you make useful modifications in your own copy, please send them back as a
pull request so others can benefit.

## Attribution

Contributors are credited through git history and may also be listed in
`AUTHORS.md` and `NOTICE`.
